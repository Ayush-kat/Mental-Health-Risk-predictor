An interactive web application using Streamlit to predict mental health risk levels based on daily lifestyle factors.

## Features

- Real-time mental health risk assessment
- 6 lifestyle parameters analysis (sleep, exercise, stress, social activity, work hours, screen time)
- Historical prediction tracking
- Interactive visualization

## Technologies Used

- **Python** - Core programming language
- **Streamlit** - Web application framework
- **Pandas & NumPy** - Data processing
- **Scikit-learn** - Machine learning implementation

## Installation

1. Clone the repository </br>
git clone https://github.com/Ayush-kat/Mental-Health-Risk-predictor.git

2. Navigate to project directory </br>
cd Mental-Health-Risk-predictor

3. Install dependencies </br>
pip install -r requirements.txt

3. Start backend </br>
python3 main.py

4. Run the application </br>
streamlit run app.py

## Usage

1. Adjust the lifestyle factor sliders
2. Click "Predict Risk" button
3. View your mental health risk assessment
4. Check historical predictions in the table
